Virtual Trial Room for Indian Dresses

Overview: The Virtual Trial Room is an innovative platform designed to provide users with a seamless and immersive experience for trying on traditional Indian dresses virtually. By leveraging AWS services, this project ensures a scalable, secure, and top-notch user experience.


Key Features
  • Realistic Dress Trials: Simulates how Indian dresses look and fit in a virtual environment.
  • Wide Range of Attires: Includes sarees, lehengas, kurtas, and more.
  • Personalized Recommendations: AI-powered suggestions based on user preferences and body type.

Technology Stack
  • AWS S3: For storing images, 3D models, and assets.
  • AWS EC2: Hosting applications and running computations.
  • AWS SageMaker: Training and deploying machine learning models.

To set up a virtual trial room, created an S3 bucket using the AWS Management Console to store datasets, images, and trained models. Configured the correct permissions for secure access.
Next, launched an EC2 instance, choosing an appropriate AMI and instance type, and configured security groups and network settings. Connected to the instance using SSH, install necessary software like Python and TensorFlow, and clone the project repository. Set up environment variables for smooth operation and secure S3 access.


Store datasets, images, and models in the S3 bucket, using AWS SDKs for easy access and management. Preprocessed the data, divide it into training, validation, and test sets, and use SageMaker for model training.
SageMaker offers a scalable environment for training, and monitoring progress ensures effective learning. Once training is complete, save the trained models to the S3 bucket and deploy them using SageMaker endpoints for real-time inference, allowing the virtual trial room to provide immediate results.


Continuous maintenance is crucial for smooth operation. Used AWS CloudWatch to monitor performance and health, and regularly update models and application code to improve accuracy and add features. Implemented robust security measures to protect user data and ensure compliance with regulations.
This approach ensures the virtual trial room operates efficiently, delivering a seamless and secure user experience.
